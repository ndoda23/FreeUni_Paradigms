# FreeUni Programming Paradigms (C)

This repository contains solutions for the Programming Paradigms course at Free University of Tbilisi, based on the Stanford CS107 course.

### Key Concepts Covered:

* **Memory Management:** Deep dive into manual memory allocation (`malloc`, `realloc`, `free`), understanding the heap vs. stack, and ensuring memory safety.
* **Generics in C:** Implementing type-independent data structures and algorithms using `void *`, size-based pointer arithmetic, and callback functions.
* **Concurrency & Multithreading:** Managing parallel execution, synchronization primitives (Mutexes, Semaphores), and handling critical issues like **Deadlocks** and **Race Conditions**.
* **x86-64 Assembly:** Analyzing how high-level C constructs (loops, function calls, structs) are translated into machine code and managed via the runtime stack.
* **Functional Programming (Scheme):** Exploring recursion, high-order functions, and the "Code as Data" philosophy using the Scheme dialect of Lisp.
